Installation Guide:


1. A phpmyadmin database connection is required. Because I am using MAMP I connected my phpmyadmin from there. 
2. If using MAMP application should be in the HTDOCS folder. 
3. In the .env change the DB to match the host, port, database, username and password to your phpmyadmin database. 
4. in terminal, cd to your app folder in HTDOCS and run the commands;
composer install
php artisan migragte
php artisan serve  
5.  Go to the URL to access the application.
6. Make sure MAMP server is runnning
